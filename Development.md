# 开发计划

## **注意事项**

- 针对每一个组件开一个分支进行开发，开发完后提 Merge Request 给 Review 后才能合并。分支名格式 `component/button`，Bug Fix 分支名格式 `fix/button`。
- 开发之前先研究别的开源项目组件 API，然后设计好组件 API，Review 后开始开发编码。
- 实现多参考已有开源项目的实现。
- 必须写单元测试。
